# cactus_pet

This is my first personal project made primarily for Boot.dev,
but also as a first project for learning game design and gamedev.
A very simple cactus virtual pet game made using Pico-8.
You can take care of a cactus pet by:

    - (rain option) watering it to heal HP (heart bar)

    - (taling face option) talking to it to increase it's happiness (smile bar)
    t
    - (soap option) washing it to increase cleanliness (soap bar)

Health, happiness, and cleanliness, decay over time. When happiness drops enough the pet will go into a sad state which will cause health to drop at an increased rate. When cleanliness drops enough the pet will go into a dirty state and health will drop at an increased rate. The sad and dirty state stack, increasing the health drop rate further. Wash and talk to the pet to recover happiness and cleanliness, removing thier respective consequences. Water the pet to increase health.
Goal: Keep the pet alive

CONTROLS:

'x' to select options

arrow keys to move cursor

TO GET STARTED:

You can find the zip files in `cactus_pet.bin`. Choose the version for
your system.

cactus_pet_oxs.zip

cactus_pet_raspi.zip

cactus_pet_windows.zip

Download and run `catus_pet.exe` to run game.
